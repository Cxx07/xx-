# 02 校内办事流程

维护人：第1组开发小组
更新日期：2026-07-15
数据来源：教务处、学生处官网通知

## 在校生业务

### 在读证明办理
- 办理地点：02A教学楼学生事务大厅
- 所需材料：本人身份证、校园一卡通
- 办结时间：当场打印盖章

### 校园卡补办
- 自助补卡机位于02A大厅，携带身份证，工本费20元 ⚠ 以官方为准

### 转专业申请
- 每年5月开放系统报名，每名学生仅可申请一次，填报2个专业志愿
- 受限人群：专升本、艺术体育类、中外合作办学不可转专业

## 教师业务

### 差旅报销
- 依据《郑州航院差旅费管理办法》，提交机票、住宿发票、出差审批单至财务处

### 调课申请
- 登录教务管理系统提交调课申请，学院教学办审核通过生效

### 教学设备报修
- 联系后勤设备科：0371-6191xxxx ⚠ 以官方为准
